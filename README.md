# Statistical Plots and Distribution Analysis

## 📊 Project Overview
This project focuses on analyzing datasets using different statistical plots and understanding data distribution. It helps to visualize data patterns, trends, and relationships using graphs and charts.

## 🎯 Objective
The main objective of this project is to:
- Perform statistical analysis on datasets
- Generate meaningful visualizations
- Understand data distribution clearly
- Help in decision making using graphs

## 📁 Features
- Data loading and cleaning
- Mean, median, mode calculation
- Histogram and box plots
- Distribution analysis
- Correlation analysis
- Graphical visualization using Python libraries

## 🛠️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Types of Visualizations Used
- Histogram
- Box Plot
- Bar Chart
- Scatter Plot
- Density Plot

```bash
# Step 1: Clone the repository

# Step 2: Go to project folder
